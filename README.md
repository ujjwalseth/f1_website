# 🏎️ F1 Driver Profile Website

A responsive portfolio-style website built from scratch, inspired by the profile of Formula 1 driver Lewis Hamilton.

##  Overview
This project is a frontend development exercise focused on building a clean, structured, and responsive website without using templates or frameworks.  
It demonstrates UI design, layout structuring, and responsive behavior across devices.

##  Objective
- Practice frontend development from scratch  
- Build a responsive and visually structured UI  
- Improve layout design and component organization  

## Tech Stack
- HTML  
- CSS  
- JavaScript  

##  Features
- Fully responsive design (mobile, tablet, desktop)  
- Structured sections (hero, profile, achievements, etc.)  
- Clean UI with consistent spacing and alignment  
- Smooth navigation and layout transitions  

##  Design Focus
- Visual hierarchy and readability  
- Consistent layout and spacing  
- Simple and intuitive user experience
